## Using blueprints with your agent

Once your agent has access to the blueprints directory, you can ask things like:

> "Check the pricing blueprint and help me structure our monetisation model"

> "I'm starting a new PM role next week — pull up the First 90 Days playbook and make a plan with me"

> "Council this: should we go freemium or free trial? Pull in the PLG blueprint for context"

The blueprints are plain Markdown — they work in Obsidian, Notion, Linear, or any text editor.
